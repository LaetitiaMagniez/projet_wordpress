Pour utiliser Le plugin Lm_Projet: Il faut créer 3 articles, chacun comportant un shortcode. Le premier devra comporter le shortcode [INSCRIPTION] qui permet de réaliser la première étape du formulaire de choix de voyage, l'inscription du prospect. Le second devra comporter le shortcode [CHOIX_PAYS] qui permet de choisir les pays souhaités. Le dernier devra comporter le shortcode [FORMULAIRE_FINAL] qui fait le récapitulatif des informations du prospect et des pays qu'il a selectionné. Dans le code du fichier Lm_Projet_Front.js, vous devez modifier "http://localhost/wordpress/2023/03/27/choix-de-pays/" à la ligne 35 par l'url du second article que vous avez créé. Veuillez rajouter ?id="+rs comme sur la ligne déjà présente pour que le plugin fonctionne correctement. Vous devez également modifier l'url "http://localhost/wordpress/2023/04/03/formulaire-final/" à la ligne 93 par l'url du dernier article que vous avez créé. Veuillez rajouter ?id="+userid comme sur la ligne déjà présente pour que le plugin fonctionne correctement. Ensuite pour procéder à l'inscription d'un nouveau prospect, rendez-vous sur le premier article que vous avez créé.
